# dada2-rnotebooks


## Requirements

The notebooks have only been tested in R under Linux (Ubuntu and Rocky Linux) with RStudio.

- R
  - `dada2`
  - `ShortRead`
  - `magrittr`
  - `Biostrings`
  - `tibble`
  - `dplyr`
  - `tidyr`
  - `stringr`
  - `phylotools`
  - `phyloseq`
- [Cutadapt](https://github.com/marcelm/cutadapt/)
- [CD-HIT](https://github.com/weizhongli/cdhit)
